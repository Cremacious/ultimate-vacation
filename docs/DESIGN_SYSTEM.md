# Design System Direction

This is not a final visual spec. It is the set of early design rules that keep the product coherent while we build.

## Design Goals

- premium but not pretentious
- playful but still functional
- minimal, airy, and easy to scan
- slightly glassy with restrained depth
- a little sassy in copy, never messy in layout

## Visual Thesis

Travel planning should feel like a polished control panel with warmth, optimism, and a bit of attitude.

## Aesthetic Rules

- use layered surfaces, not clutter
- prefer soft gradients, depth, and blur over heavy borders everywhere
- keep color count tight
- let typography and spacing do most of the work
- use cards only when they create real interaction clarity

## Color Direction

Current direction:

- warm off-white base
- deep slate text and UI anchors
- sky and sea-inspired cool accents
- sun-warmed amber highlights

Avoid:

- default purple SaaS palettes
- overly dark travel UI by default
- harsh neon styling

## Typography Direction

- strong headline hierarchy
- clean sans-serif for product UI
- compact uppercase utility labels where helpful
- no overly decorative travel fonts

## Surface Strategy

- hero and marketing surfaces can be more atmospheric
- app surfaces should be calmer and more operational
- glass effects should be subtle and readable
- shadows should suggest depth, not fog

## Motion Direction

- soft entrance motion
- small hover lift on interactive surfaces
- phase transitions should feel smooth and directional
- motion should help orientation, not distract from tasks

## Copy Direction

### Brand tone

- smart
- cheeky
- capable
- reassuring

### Copy rules

- headlines can have attitude
- product instructions must stay clear
- empty states should feel helpful and lightly witty
- avoid trying too hard to sound funny

## UX Rules

- show what matters now
- keep primary actions obvious
- reduce decision fatigue
- expose advanced features progressively
- make collaboration feel guided, not overwhelming

## Accessibility Principles

- maintain strong contrast even with translucent surfaces
- avoid using glass effects as a replacement for structure
- touch targets must be generous
- motion should remain subtle and tolerable
- important checklist states should never depend on color alone
